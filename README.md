# LEAD-MANAGER-PRO-DESKTOP
# 🚀 Lead Manager Pro - Ex-Car Edition

O **Lead Manager Pro** é uma aplicação web progressiva (PWA) de alta performance, projetada para a gestão ágil de leads e vendas. Com uma interface inspirada no design minimalista do ecossistema Apple, o sistema oferece uma experiência fluida para consultores e administradores.

## 📱 Funcionalidades Principais

- **Registro Inteligente de Leads:** Cadastro rápido com detecção automática de DDD e validação de duplicidade.
- **Gestão de Vendas (Leads Verdes):** Módulo específico para conversão de vendas com acompanhamento em tempo real.
- **Painel de Metas Dinâmico:** Monitoramento visual do progresso mensal com barra de porcentagem automática.
- **Relatórios Profissionais em PDF:** Geração de relatórios detalhados, separados por consultor, com cabeçalhos padronizados e totalizadores automáticos.
- **Segurança Administrativa:** Funções críticas (exclusão, reset e gerenciamento) protegidas por senha de administrador.
- **PWA (Progressive Web App):** Instalável em PCs e Celulares, permitindo acesso rápido diretamente da tela inicial.
- **Offline First:** Persistência de dados via `localStorage` e suporte a Service Worker para maior estabilidade.

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3 (Glassmorphism UI), JavaScript Vanilla.
* **Gráficos:** [Chart.js](https://www.chartjs.org/) para visualização de dados.
* **Documentação:** [jsPDF](https://github.com/parallax/jsPDF) e [AutoTable](https://github.com/simonbengtsson/jsPDF-AutoTable) para relatórios.
* **PWA:** Web Manifest e Service Workers para instalação nativa.

## 📦 Como Instalar

Como este projeto é um PWA, você pode instalá-lo seguindo estes passos:

1.  Acesse o link do projeto via **Google Chrome** ou **Microsoft Edge**.
2.  No PC: Clique no ícone de instalação (computador com seta) na barra de endereços.
3.  No Celular: Vá em "Configurações" do navegador e selecione **"Adicionar à tela de início"** ou **"Instalar Aplicativo"**.

## 🔐 Configurações de Administrador

O sistema possui uma camada de proteção para gerenciamento de dados:
- **Senha Padrão:** `Hudson@2026`
- **Backup:** Funções de Exportar e Importar banco de dados em formato JSON incluídas no painel lateral.
